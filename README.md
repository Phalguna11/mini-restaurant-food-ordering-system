# Restaurant Food Ordering System

## Project Overview
This is a console-based Restaurant Food Ordering System developed using Python. It allows users to view the menu, place orders, and generate a final bill.

## Features
- Categorized menu display
- Multiple item ordering
- Quantity selection
- Total bill calculation
- Input validation
- Structured program design

## Technologies Used
- Python
- Dictionaries
- Functions
- Loops
- Conditional statements

## How to Run
1. Install Python.
2. Open terminal in the project folder.
3. Run:
   python restaurant_ordering_system.py
